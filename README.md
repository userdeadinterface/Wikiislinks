# Wikiislinks
Gets link for Icelandic Wikipedia Page from keyword.

## Install:
Install here: https://github.com/userdeadinterface/Wikiislinks/releases/tag/v1.0.0

## Use:
.. code-block:: python
    from wikicode import WikisGetLink
    WikisGetLink("Python")

    
## Output:

    Python    
    _________    
    https://is.wikipedia.org/wiki/Python
